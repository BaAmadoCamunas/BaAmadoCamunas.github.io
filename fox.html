Fox

Follows - FK/IK System - Non Roll - Stretch/Squash

3D model from H.N., ArtStation.



This quadruped rig has been designed with a focus on flexibility, control and real-time compatibility, making it ideal for video game environments.



Facial Rig

The facial configuration uses a bone-based deformation system only (without corrective shapes) allowing optimal performance for game engines. It includes primary and secondary controls for more expressive animations. In addition, the ears incorporate an FK/IK system to improve the flexibility of ear poses and give the animator more control over the ears.


To implement this video game-oriented facial rig, a two-stage process was followed:

  • Blocking (first image): in this initial stage, the weights of the mesh vertices were assigned to the corresponding bones, establishing a solid base for the deformation.

  • Smooth (second image): subsequently, smoothing of weights between adjacent bones was performed in order to achieve a more natural transition and smoother deformation in facial expressions.

Body Rig

The body rig integrates multiple FK/IK systems for the spine, legs and tail, offering smooth transitions and versatile animation possibilities. For the front legs, a combination of Rotate Plane Solver and Single Chain IK Handles have been used to improve the control of this area and for the hind legs, a Spring Solver has been combined with a Rotate Plane Solver, simulating a more dynamic movement.

The Stretch and Squash systems have been applied to the spine, tail and limbs, allowing the animator to control the stretchiness value through an attribute.


In order to implement this body rig, it has been necessary to follow the following process:

  • Blocking (first image): in this step, the weights corresponding to each vertex of the mesh have been given.

  • Smooth (second image): in this stage, the weights have been distributed among the bones to achieve a better result.
