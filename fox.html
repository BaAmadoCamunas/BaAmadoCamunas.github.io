3D model from H.N., ArtStation.



This quadruped rig was designed to provide animators with precise control, flexibility and real-time compatibility, making it ideal for video game and animation workflows.



Facial Rig

The facial rig uses a bone-based deformation system only (without corrective shapes), ensuring optimal performance for game engines. It includes primary and secondary controls for expressive animations, while the ears feature an FK/IK system to allow more flexible poses and greater animator control.

See on Vimeo

To implement this video game-oriented facial rig, a two-stage process was followed:

  • Blocking (first image): weights were assigned to each bone, establishing a solid deformation base.
  • Smooth (second image): subsequently, weights were smoothed between adjacent bones for natural transitions and improved facial deformation.

Body Rig

The body rig integrates multiple FK/IK systems for the spine, legs and tail, providing smooth transitions and versatile animation possibilities.

Front legs use a combination of Rotate Plane Solver and Single Chain IK Handles to improve control, while hind legs employ a Spring Solver combined with a Rotate Plane Solver to simulate dynamic movement.

Stretch and Squash systems were applied to the spine, tail and limbs, allowing animators to adjust the stretchiness via a dedicated attribute.

See on Vimeo

In order to implement this body rig, it has been necessary to follow the following process:

  • Blocking (first image): initial weights were assigned to each vertex on the mesh.

  • Smooth (second image): weights were smoothed across bones for more natural deformation.
