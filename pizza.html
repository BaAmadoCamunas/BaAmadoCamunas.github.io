Pizza

Plane System · FK Chain · Deformers

See on Vimeo


This rig allows animators to create flexible cartoon-style pizza deformations, with precise control over edge deformation, curvature and dynamic bending.



Deformation

The rig uses Sine and Bend deformers, along with Soft Mode, to give the animator greater control over the pizza's shape. An FK chain was also implemented to allow realistic curvature and flexible animation.

Plane System

A plane system has been created to allow cartoon-style deformations, giving the animator greater control over edge bending and exaggerated shapes.
